# Guía Rápida: n8n + Wasapi

Instala n8n localmente y configura el paquete de nodos de Wasapi para automatizar WhatsApp en pocos pasos.

## ¿Qué es n8n?

n8n es una plataforma de automatización visual que te permite conectar servicios y crear workflows sin código.

## Instalación Rápida

### 1. Instalar n8n

```bash
npm install n8n -g
n8n --version
```

<TerminalMock />

### 2. Ejecutar n8n

```bash
n8n
```

Accede a: `http://localhost:5678`

### 4. Instalar Wasapi en N8n

ve a a seccion ajustes de comunidad de nodes e instala el paquete de Wasapi
la url es http://localhost:5678/settings/community-nodes

copia el paquete aqui
```
@laiyon/n8n-nodes-wasapi@latest
```

### 5. Credenciales de Wasapi

1. Crea una credencial de Wasapi 
2. Ingresa la API Key de Wasapi en n8n
la url es http://localhost:5678/home/credentials
